O que é o Disco de um computador?



Um disco de computador, também conhecido como unidade de armazenamento, é um dispositivo responsável por armazenar dados de forma permanente ou semi-permanente em um computador. Ele guarda todos os arquivos, documentos, programas, e o próprio sistema operacional, permitindo que esses dados sejam acessados e utilizados quando necessário.


Qual sua funcão?


Armazenar informações. Quando você salva um arquivo, instala um programa ou baixa algo da internet, todos esses dados são armazenados no disco. Ele também guarda o sistema operacional, que é essencial para que o computador funcione e inicie.



Tipos de Disco


HDD (Hard Disk Drive): Usa discos magnéticos para armazenar informações. Ele possui partes móveis, como pratos e braços de leitura, e é utilizado para armazenar grandes quantidades de dados.

SSD (Solid State Drive): Armazena dados em chips de memória flash, não tem partes móveis, e é mais rápido e resistente do que os HDDs, sendo uma escolha comum em computadores modernos.

NVME (Memória expressa não volátil): é um protocolo para transferência de dados altamente paralela com sobrecarga de sistema reduzida por entrada/saída (E/S) que é usada no armazenamento flash e unidades de estado sólido (SSDs.)
